## <a name="quick-start"></a>Quick Start

Feel free to use npm or yarn to run all scripts provided in this source code.

1. **Install via npm or yarn:**

  Using npm:
```bash
npm install
```
  Using yarn:
```bash
yarn
```

2. **Run TSLint:**
```bash
yarn test
```

3. **Offline development server:**
```bash
yarn start-offline
```

4. **Deploy your code to AWS:**
```bash
yarn deploy
```
